Restrict users without capability to edit others posts see other uses posts

== Description ==

By default users in WordPress can see other user's posts, if even they don't capabilities to edit those posts.
'Manage Others Posts Capabilities' is simple plugin with zero configuration, that will help you to restrict 
users without capabilities to `edit_others_posts` to see other user's posts

== Installation ==

1. Upload `manage-others-post-capabilities` folde to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!
